Essential Web and HTML concepts

How the WWW Works

The WWW is a collection of documents in the HyperText Markup Language (HTML) format. Hyperlinks are what make the web web-like. The WWW uses special computers called servers. Servers are computers that are optimized for sitting in a closet and hosting files. A browser makes requests to servers via the Internet using HTTP. HTTP is a protocol for communicating with servers.

Important HTML Concepts

HTML consists of:

  • Text content (what you see)
  • Markup (what the content looks like)
  • References to other documents (e.g. images, video)
  • Links to other pages
HTML uses tags to markup a page. There are opening and closing tags. An element consists of opening tag, its content, and it's closing tag. An element has form '<tag_name tag_attribute="some_val"> content_to_display </tag_name>'. A tag does not always have an attribute. Furthermore, some elements do not have a closing tag. For instance, the '<img src="some_val" alt="some_other_val">'

The general structure of an HTML document is:

<!DOCTYPE HTML>
<html>>
<head>
<title>some_title</title>
</head>
<body>
</body>
</html>
